SQL subsets
-----------
DDL data definition language
---------------
# provides cmds for defining, deleting, modify tables
#cmd used to create storage objects in db, like tables

>> CREATE TABLE table_name_wanted (column_name1 datatype(size), column_name2 datatype(size), column_name3 datatype(size));

#cmd used to modify the structure of a table in db
>> ALTER TABLE table_name ADD (column_name datatype(size));

#add a primary key to a table
>> ALTER TABLE table_name ADD primary key (column_name);


#cmd used to remove a object from db
>> DROP TABLE table_name;

# remove all records from a table, but not delete table
>> TRUNCATE TABLE table_name;

# comment cmd in SQL -> using double dash --
--Retrieve all data from a table
SELECT * FROM table_name


DML data manipulation language
--------------------
#cmd used to insert records of data into db
>> INSERT INTO table_name (column1, column2, column3) VALUES (value1, value2, value3);

#cmd used to edit data already in db
>> UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition;

#cmd used to delte one or more rows of data from a table
>> DELETE FROM table_name WHERE condition;

DQL data query language
-------------------
Used to select data stored in a database
>> SELECT * FROM table_name;

DCL data control language
--------------------
Used to control access to the db
>> ddl grant
>> ddl revoke

TCL transaction control language
----------------------
used to manage transactions in db,
#cmd to save all work have done in db
>> COMMIT

#used to restore a db to a previous commited state
>> ROLLBACK
