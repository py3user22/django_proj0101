SQL subsets
-----------
DDL data definition language
---------------
#cmd used to create storage objects in db, like tables
>> DDL create
#cmd used to modify the structure of a table in db
>> DDL alter
#cmd used to remove a object from db
>> DDL drop


DML data manipulation language
--------------------
#cmd used to insert records of data into db
>> DML insert

#cmd used to edit data already in db
>> DML update

#cmd used to delte one or more rows of data from a table
>> DML delete

DQL data query language
-------------------
Used to read data stored in a database

DCL data control language
--------------------
Used to control access to the db
>> ddl grant
>> ddl revoke
